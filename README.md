# 📌 Project Overview

A simple and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.

Two players take turns marking **X** and **O** on a **3×3 grid** until one player wins or the game ends in a draw.

# 🎮 How to Play

1. Player **O** starts the game
2. Players take turns placing **X** and **O**
3. The game automatically checks winning patterns
4. If a player wins → Winner message appears
5. If all boxes are filled with no winner → **Draw**
